# Getting started nuxt
Nuxt.js 처음부터 시작하기

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm run dev
```

## Work contents

#### 1. install
```
npm install nuxt --save
```

#### 2. Create index.vue
/pages/index.vue
```
<template>
    <h1>Hello world!</h1>
</template>

<script>
    export default {
        name: "index"
    }
</script>
```    
#### 3. npm run dev
```
npm run dev
```
